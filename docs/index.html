Smart Greenhouse

Hệ thống IoT nhà kính thông minh — đo nhiệt độ, độ ẩm, ánh sáng và tự động điều khiển đèn + cửa thông gió. Đồ án cuối kỳ IAD591.

Arduino Uno Raspberry Pi Flask + SQLite Dashboard realtime
🔧 Bắt đầu lắp ráp 🔌 Sơ đồ đấu nối tương tác

Hệ thống làm gì?

Mô phỏng một nhà kính tự động: cảm biến đo môi trường → tự bật đèn khi tối, mở cửa khi nóng → hiển thị & điều khiển trên dashboard web.

📡

Thu thập

DHT22 đo nhiệt/ẩm, LDR đo ánh sáng — đọc liên tục mỗi giây.

🤖

Tự động

Tối → bật đèn trồng cây. Nóng → mở cửa thông gió (servo). Có chế độ Auto/Manual.

📊

Giám sát

Dashboard web: số liệu realtime, biểu đồ lịch sử, điều khiển từ xa, lưu SQLite.

Kiến trúc 3 tầng

Đúng mô hình IoT của đề bài. Điểm mấu chốt: Arduino nối Pi bằng cáp USB — không dùng dây TX/RX, không cần hạ áp, hết lỗi truyền dữ liệu.

Lắp ráp từng bước

Quy tắc vàng: lắp tới đâu test tới đó. Xong Layer 1 = đã chắc ~8 điểm. Layer 2 làm nếu còn giờ — mỗi phần độc lập, hỏng không ảnh hưởng phần khác.

🟢 LAYER 1 — Lõi an toàn (mục tiêu ~8 điểm)
  1. Lắp DHT22 → D2
    ✓ PASS: Serial Monitor (9600) thấy JSON có temp, hum ≠ 0
  2. Lắp LDR → A0 (kèm trở 10kΩ)
    ✓ PASS: che/rọi đèn → giá trị light đổi rõ → ghi lại đặt ngưỡng
  3. Lắp LED → D7 (kèm trở 220Ω)
    ✓ PASS: gõ LED:1 → sáng, LED:0 → tắt
  4. Cắm cáp USB Arduino → Raspberry Pi
    ✓ PASS: trên Pi chạy ls /dev/ttyACM* thấy ttyACM0
  5. Chạy Flask, mở dashboard
    ✓ PASS: pill "Live" xanh, biểu đồ chạy, gạt công tắc → LED thật bật/tắt
🟡 LAYER 2 — Tăng điểm (tuỳ chọn)
  1. Servo cửa thông gió → D9
    Nếu giật/reset → cấp nguồn riêng hoặc bỏ #define USE_SERVO
  2. LCD1602 I2C → A4/A5
    Màn trắng → vặn contrast / đổi 0x27→0x3F / bỏ #define USE_LCD
  3. Bonus phần mềm (đã có sẵn)
    Quay demo Auto↔Manual + chỉnh ngưỡng trên web

Cách chạy

1️⃣ Nạp code Arduino

Cài thư viện trong Arduino IDE: DHT sensor library (Adafruit) + LiquidCrystal_I2C. Rồi nạp file:

arduino/greenhouse/greenhouse.ino

2️⃣ Chạy trên Raspberry Pi

cd rpi
pip3 install -r requirements.txt
python3 app.py

Mở trình duyệt: http://<ip-của-pi>:5000

Dashboard trông như vầy 👇

🛡️ Chống sập demo: Không có Arduino → dashboard tự chạy chế độ Simulated, vẫn demo được biểu đồ + điều khiển.
